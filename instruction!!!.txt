ВНИМАНИЕ!!!
Сохраните этот репозитории себе на ПК.
Подготовьте виртуальное окружение (venv),
чтобы взаимодействовать со сервисом.
_________
Выполняем все пункты по порядку:

1) Запускаем в терминале виртуальное окружение:
    C:\Windows\System32\...\venv\Scripts\activate

2) Переходим по директории:
C:\...\GitHub\Winter-hackathon_SKF-MTUCI_08-12-2024\ready code

3) Запускаем файл "server.py":
    python server.py

и переходим по адресу (отобразится в строке "* Running on...")
через браузер.

4) Перед пользователем отобразится страница "системы мониторинга"

    Пользуйтесь на здоровье :-)

P.S. программа для злоумышленника (intruder.py)

но если будет интересно ознакомиться с частичной реализацией,
то выполняйте пункты с 1 по 3,
